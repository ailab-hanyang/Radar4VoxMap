# Radar4VoxMap

Radar4VoxMap is a radar odometry system that utilizes voxel mapping and point-to-covariance doppler-ICP.

Our research paper is currently under review. The source code will be made available upon publication.

## Demonstration

The demonstration is conducted on the [VoD dataset](https://github.com/tudelft-iv/view-of-delft-dataset) scenario 19 (index: 06759-07542).
